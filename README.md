Setup
1. Create a Supabase project.
2. In Supabase Settings -> API, copy the Project URL and anon key.
3. Fill these in your env file:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   - VITE_WEATHER_API_KEY (optional, for weather card)
4. In Supabase Auth, create the users you want to sign in with.
5. Run 2505-frontend/supabase/seed.sql in the Supabase SQL Editor.
